How to Add SEEP to Your Shopify Store

  1. Log into your Shopify Admin
  2. Go to Online Store → Themes
  3. Click Actions → Edit Code
  4. Under the Layout folder, open theme.liquid
  5. Scroll to the bottom of the file, just above </body>
  6. Paste the script:
<script src="https://noble-server.onrender.com/widget.js?shop=your-shop.myshopify.com" async></script>
  1. Click Save

How to Find Your Shopify Domain

Look at the URL when logged into Shopify, it’s usually:
your-store.myshopify.com

How to Get Your Storefront Access Token

  1. In Shopify Admin, go to Apps → Develop Apps
  2. Click Create App
  3. Name it “SEEP Assistant”
  4. Go to Configuration → Storefront API
  5. Add the following access scope:
    read_products
  6. Save and Install the app
  7. Shopify will generate a token — copy and paste this into SEEP
